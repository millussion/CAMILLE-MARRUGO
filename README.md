# ✦ CAMILLE MARRUGO - Personal Portfolio

This repository contains the source code for my personal portfolio. It's a space designed to showcase who I am, my creative process, and the projects I've built as a web developer and designer.

---

## Visual Concept

The portfolio follows a **modern, avant-garde aesthetic** with a focus on:
* **Typography-driven design:** Bold headers and experimental layouts.
* **Interactive elements:** Custom sidebar navigation and image modals (specifically for my pets section).
* **Motion:** Seamless text animations and transitions that reflect my creative energy.

---

## Key Sections

* **Hero / Head:** An abstract, high-impact intro asking "What is Camille doing?".
* **About Me:** A deep dive into my background, my passion for technology, and my education in Web Programming.
* **Mascotas (Pets):** A personalized interactive grid showcasing my cats, Mia and Misha, using a custom lightbox/modal system.
* **Projects:** A call-to-action section that leads to my creative work and imaginative experiments.
* **Contact & Socials:** Direct links to my GitHub, Instagram, and professional email.

---

##  Built With

* **HTML5:** Structured with semantic elements for better accessibility.
* **CSS3:** Custom styling (via `index.css`) including marquee animations, grid layouts, and responsive sidebars.
* **JavaScript:** Powering the navigation logic (show/hide sidebar) and the interactive pet gallery.
